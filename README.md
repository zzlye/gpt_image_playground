# Wenyun Workshop

## 简介

Wenyun Workshop（文运工坊）是一款面向创作流程的图片、文字、视频与画布工作台。它把提示词、参考图、素材、画布节点和历史记录放在同一个使用体验里，方便连续创作、整理素材和复用结果。

## 如何使用

1. 打开文运工坊页面。
2. 在设置里填写可用接口信息。
3. 选择文运工坊、画布工坊或视频相关功能。
4. 输入提示词，按需添加参考图或素材。
5. 生成后可以查看详情、保存结果、加入素材库或继续在画布中编排。
6. 需要迁移数据时，可在设置里使用数据管理与同步功能。

## 功能介绍

- 文运工坊：支持提示词输入、参考图添加、模型选择、历史查看和素材管理。
- 画布工坊：支持节点编排、图片节点、文字节点、视频节点、连线、分组、裁剪和资产整理。
- 素材库：支持上传、预览、重命名、删除、标签分类和在生成流程中引用。
- 模型列表：区分不同站点模型，展示可用模型与对应价格。
- 数据管理：支持选择性导出、导入和同步文运工坊、画布工坊与素材库数据。
- 深色与浅色外观：界面会跟随当前主题调整卡片、弹窗和删除确认等样式。

## 使用说明

- API 与配置属于本地使用数据，请自行妥善保存。
- 生成内容与素材建议定期导出备份。
- 如果需要在多台设备间使用，请先在设置中配置同步范围，再执行上传或拉取。
